# Production Checklist

Use this before, during, and after recording the MindsHub ad segment.

## Before Recording

- [ ] MindsHub desktop app installed and up to date
- [ ] GitHub connected in MindsHub (check Settings > Connected Apps)
- [ ] A Rust repo with open issues is accessible (real or use `demo-data/`)
- [ ] Script printed or open on second monitor (`script/ad-script.md`)
- [ ] OBS / screen recorder running, mic levels checked
- [ ] Notifications silenced (OS + Slack + email)
- [ ] Browser tabs closed except what you need
- [ ] Resolution: 1080p or higher recommended

## During Recording

- [ ] Open MindsHub first — let the UI breathe on screen for 2 seconds
- [ ] Show Connected Apps panel (GitHub must be visibly listed)
- [ ] Type the demo prompt at natural speed — don't rush
- [ ] **Let the activity stream run without cutting** — 3–4 seconds of silence
- [ ] Hold the final markdown output on screen for at least 4 seconds
- [ ] Say pricing clearly: *"five million tokens free... $9.95 Pro"*
- [ ] End hard: *"Okay — back to Rust."*

## After Recording

- [ ] Trim to 60–70 seconds max
- [ ] Check audio: pricing line must be clearly audible
- [ ] Add to video description: `MindsHub AI → https://mindshub.ai`
- [ ] Pin `mindshub.ai` as a top comment
- [ ] Add chapter marker in YouTube for the sponsor segment (optional)
